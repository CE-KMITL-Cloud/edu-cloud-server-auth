## Installation

```bash
> yarn install
```

## Running the app

#### run postgresql database

```bash
> cd edu-cloud-api
> docker-compose up
```

#### run radis

```bash
> docker-compose up
```

#### pull database

```bash
> yarn db-pull:dev
```

#### generate prisma code from db

```bash
> yarn generate
```

#### start app

```bash
> yarn dev
```
